# Playwright MCP Workshop

A hands-on workshop for learning Playwright with Model Context Protocol (MCP).

## Overview

This workshop guides you through using Playwright for browser automation and testing, integrated with the Model Context Protocol (MCP) to enable AI-driven browser interactions.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- A GitHub Copilot subscription (for MCP integration)

## Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/CopilotLabs007/playwright-mcp-workshop.git
   cd playwright-mcp-workshop
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Install Playwright browsers**

   ```bash
   npx playwright install
   ```

4. **Run the tests**

   ```bash
   npx playwright test
   ```

## Workshop Modules

- **Module 1**: Introduction to Playwright
- **Module 2**: Writing your first test
- **Module 3**: MCP integration with GitHub Copilot
- **Module 4**: AI-assisted browser automation
